# FFmpeg Video Processing

Comprehensive FFmpeg video processing skill with practical examples for common video tasks.

## Description

基于 FFmpeg 的视频处理技能，提供最实用的日常视频处理命令示例。覆盖 90% 的日常需求，所有示例均经过验证可直接使用。

## Usage

This skill provides ready-to-use FFmpeg commands for practical scenarios:

### 1. 视频转码（全设备兼容）
- MKV/AVI/WebM/MOV 转 MP4
- H.264 + AAC 编码，所有设备都能播放

### 2. 视频压缩
- 压缩到 10MB 以下（邮件/微信发送）
- H.265/HEVC 高效编码，文件更小

### 3. 视频裁剪
- 去掉片头片尾
- 按时间精确裁剪

### 4. 视频拼接
- 合并多个视频片段
- 使用 concat 协议快速拼接

### 5. 提取音频
- 做手机铃声/背景音乐
- 提取为 MP3/WAV/AAC

### 6. 添加水印
- 版权保护/品牌标识
- 图片/文字水印

### 7. 视频转 GIF
- 做表情包/动图
- 高质量 GIF 转换

### 8. 生成缩略图
- 视频预览/封面
- 生成缩略图网格

### 9. 旋转/翻转视频
- 手机拍摄方向不对
- 旋转、镜像翻转

### 10. 调整音量
- 声音太小/太大
- 自动归一化音量

### 11. 移除音频
- 制作无声视频

### 12. 替换音频
- 给视频换背景音乐

### 13. 分辨率调整
- 适配不同屏幕
- 4K/1080p/720p/480p

### 14. 帧率调整
- 慢动作/快动作

### 15. 批量处理
- 多个视频一起处理

### 16. 查看视频信息
- 了解视频参数

### 17. 常见问题修复
- 音视频不同步
- 修复损坏的视频
