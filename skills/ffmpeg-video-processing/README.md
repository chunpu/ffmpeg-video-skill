# FFmpeg Video Processing

Comprehensive FFmpeg video processing skill with practical examples for common video tasks.

## Description

基于 FFmpeg 的视频处理技能，提供格式转换、压缩、裁剪、拼接、音频提取、添加滤镜等典型场景的完整命令示例。覆盖 90% 的日常视频处理需求，所有示例均经过验证可直接使用。

## Usage

This skill provides ready-to-use FFmpeg commands for:

### 1. 格式转换 (Format Conversion)
- MP4 ↔ WebM ↔ MOV ↔ AVI 等格式互转
- 无损或接近无损编码转换
- 快速流复制转换

### 2. 视频压缩 (Video Compression)
- CRF 恒定质量压缩
- H.265/HEVC 高效编码
- 硬件加速压缩

### 3. 分辨率调整 (Resolution Scaling)
- 1080p/720p/480p 等常用分辨率
- 按比例或固定尺寸缩放
- 保持宽高比适配

### 4. 视频裁剪 (Video Trimming)
- 按时间精确裁剪
- 按帧裁剪
- 快速流复制裁剪

### 5. 视频拼接 (Video Merging)
- 使用 concat 协议拼接
- 重新编码拼接
- 批量视频合并

### 6. 音频处理 (Audio Processing)
- 提取音频 (MP3/WAV/AAC)
- 替换音频轨道
- 调整音量
- 移除音频

### 7. 视频滤镜与特效 (Video Filters & Effects)
- 旋转、翻转、镜像
- 亮度、对比度、饱和度调整
- 模糊、去水印
- 画面裁剪

### 8. 添加文字与水印 (Text & Watermark)
- 添加文字叠加
- 添加图片水印
- 半透明水印
- 动态时间戳

### 9. 视频转 GIF (Video to GIF)
- 高质量 GIF 转换
- 两步优化转换
- 裁剪 GIF

### 10. 视频预览与缩略图 (Thumbnails & Previews)
- 生成单张缩略图
- 批量生成缩略图
- 生成缩略图网格预览

### 11. 帧率调整 (Frame Rate)
- 修改帧率
- 平滑插帧

### 12. 视频信息查看 (Video Information)
- 查看元数据
- 获取时长、分辨率等信息

### 13. 批量处理 (Batch Processing)
- Shell 脚本批量处理示例

### 14. 高级技巧 (Advanced Tips)
- 硬件加速
- 流式处理
- 预设文件

### 15. 常见问题与解决方案
- 音视频同步修复
- 损坏视频修复
- 播放顺序问题
